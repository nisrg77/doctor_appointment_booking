================================================================================
  ENVIRONMENT VARIABLES REQUIRED
================================================================================

  — appointment-backend/.env —
    PORT=5000
    MONGO_URI=mongodb+srv://<user>:<password>@cluster.mongodb.net/<dbname>
    JWT_SECRET=<a_long_random_secret_string>

  — appointment-frontend/.env —
    VITE_API_URL=http://localhost:5000/api

  ⚠ WARNING: Never commit .env files to Git. They are excluded via .gitignore.


================================================================================
  HOW TO INSTALL EVERYTHING FROM SCRATCH
================================================================================

  Step 1 — Clone the repo:
    git clone https://github.com/nisrg77/doctor_appointment_booking.git
    cd doctor_appointment_booking

  Step 2 — Install root dependencies:
    npm install

  Step 3 — Install backend dependencies:
    cd appointment-backend
    npm install
    cd ..

  Step 4 — Install frontend dependencies:
    cd appointment-frontend
    npm install
    cd ..

  Step 5 — Create .env files:
    • Copy the variables listed above into appointment-backend/.env
    • Copy the variables listed above into appointment-frontend/.env

  Step 6 — Run the full stack in development mode:
    npm run dev        (from the root d:\MERN\ directory)

  The backend will start on  →  http://localhost:5000
  The frontend will start on →  http://localhost:5173
